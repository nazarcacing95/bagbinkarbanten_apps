Sistem Terintegrasi Biro SDM Polda Banten

Sistem Terintegrasi Biro SDM Polda Banten adalah sebuah portal dashboard modern yang dirancang untuk mengonsolidasikan berbagai aplikasi administrasi personil dalam satu antarmuka yang elegan dan responsif. Dashboard ini berfungsi sebagai hub utama untuk mengakses alat penghitung UKP, validator data mutasi, dan generator indikator profesionalitas.

ğŸ“– Cara Penggunaan

Pastikan semua file modul (index.html, DEVELOPMENT.HTML, INDIKATOR.HTML) berada dalam folder yang sama dengan file dashboard utama.

Buka file dashboard utama (misal: dashboard.html) melalui browser modern (Chrome, Edge, atau Safari).

Gunakan menu di sebelah kiri (Sidebar) untuk berpindah antar aplikasi.

Pada perangkat mobile, klik ikon bar di pojok kiri atas untuk membuka menu navigasi.

ğŸ“ Catatan Pengembangan

Sistem ini menggunakan allowtransparency="true" pada elemen iframe untuk memastikan latar belakang partikel dashboard tetap terlihat di balik aplikasi yang dimuat.

Untuk performa optimal, disarankan agar file-file anak (sub-apps) juga menggunakan skema warna yang senada (transparan atau semi-transparan).
